# Customization

FlyPage is highly customizable. You can adjust the colors, layout, and components to match your brand.

## Theme Colors

All colors are defined as CSS variables in `src/App.css`.

### Dark Mode Palette

| Variable | Description | Value |
| --- | --- | --- |
| `--bg-primary` | Background color | #121212 |
| `--bg-secondary` | Sidebar/Header background | #1e1e1e |
| `--accent-color` | Primary accent color | #bb86fc |
| `--text-main` | Main text color | #e1e1e1 |

## Customizing Components

You can find all UI components in the `src/components/` directory. Each component is written in TypeScript and styled with Vanilla CSS.
