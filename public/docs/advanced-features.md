# Advanced Features

Explore the advanced capabilities of the Simple Docs platform, including custom callouts, task lists, and more.

## Callouts (Admonitions)

Callouts help highlight important information, warnings, or tips. In this template, we style blockquotes differently depending on their content.

> [!INFO]
> This is an information callout. Use it to provide additional context or helpful tips that are not critical but useful.

> [!WARNING]
> This is a warning callout. Use it to alert users about potential pitfalls or things to be cautious about.

> [!ERROR]
> This is an error callout. Use it for critical information, breaking changes, or dangerous actions.

> [!SUCCESS]
> This is a success callout. Use it to indicate completed steps or positive outcomes.
